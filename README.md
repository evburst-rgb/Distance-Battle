# Distance Battle

## Every Meter Matters.

Distance Battle is a custom Rocket League game mode created by EvBurst.

Players must complete every distance from **1–200** while avoiding the **Dead Zone (201–249)** before finishing with a **250+** shot.

## Features

- Flight Distance Tracking
- Live HUD
- Progress Board
- Automatic Save System
- Reset Progress
- Custom Arena
- Cinematic Camera
- Electric Blue Energy Gate
